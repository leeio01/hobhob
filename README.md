## 📜 Contract Overview

The contract is a simplified version of the TRC-20 standard, featuring:

- `transfer()`
- `approve()`
- `transferFrom()`
- `allowance()`
- `balanceOf()`
- Events: `Transfer`, `Approval`

No external dependencies are used, and all logic is contained in a single Solidity file.
